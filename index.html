Forking or installing this repo

Either use Git to download this repo so you can work on it:

# Clone the repo
git clone git@github.com:hash-bang/ngCookbook.git

# Install all bower components
bower install

Or fork this repo and work on it yourself.

Recipe rules

All ngCookbook examples must conform to the following standards:

  • Each example must occupy its own folder - including image, css, json or other assets
  • Each example should be as minimal and to the point as possible
  • Each example should endevour to be as copy/paste compatible as possible
  • Each should make adequate use of Bootstrap for layout but not use too many custom CSS or JavaScript elements that would be found outside of a regular website
  • Use the template receipe as a basis for new recipes
